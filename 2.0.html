Pasos para crear un archivo HTML:

  1. Abre el editor de texto:

  2. Copia el código:

    • En "Tipo", selecciona "Todos los archivos".

    • Elige una ubicación fácil de encontrar (por ejemplo, el escritorio) y haz clic en Guardar.

  5. Abre el archivo en el navegador:

    • Ve a la ubicación donde guardaste el archivo index.html y haz doble clic en él.

    • ¡La página se abrirá en tu navegador y podrás ver cómo se ve!

Consejos adicionales:

<script>
    let carrito = [];

    function actualizarCarrito() {
        const listaCarrito = document.querySelector('.lista-carrito');
        const totalElemento = document.querySelector('.total');

        if (carrito.length === 0) {
            listaCarrito.innerHTML = '<p>No hay productos en el carrito.</p>';
            totalElemento.textContent = 'Total: $0.00';
            return;
        }

        listaCarrito.innerHTML = '';
        let total = 0;

        carrito.forEach((producto, index) => {
            const item = document.createElement('p');
            item.textContent = `${producto.nombre} - $${producto.precio.toFixed(2)}`;
            listaCarrito.appendChild(item);
            total += producto.precio;
        });

        totalElemento.textContent = `Total: $${total.toFixed(2)}`;
    }

    function agregarAlCarrito(nombre, precio) {
        carrito.push({ nombre, precio });
        actualizarCarrito();
        alert(`${nombre} agregado al carrito!`);
    }

    function vaciarCarrito() {
        carrito = [];
        actualizarCarrito();
        alert('El carrito ha sido vaciado.');
    }
</script>
